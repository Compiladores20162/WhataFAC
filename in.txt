IDENTIFICATION DIVISION.
PROGRAM-ID.    notas.
DATA DIVISION.
WORKING-STORAGE SECTION.
01 W-BRANCOS               PIC X(55).
01  W-ALUNO                 PIC X(50).
01  W-NOTA1                 PIC 9(03).
01  W-NOTA2                 PIC 9(03).
01  W-NOTA3                 PIC 9(03).
01  W-NOTA4                 PIC 9(03).
01  W-TOTAL-NOTAS           PIC 9(03).
01  W-MEDIA                 PIC 9(02).

PROCEDURE DIVISION.
MAIN-PROCEDURE.

DISPLAY "Informe o Aluno.:"
DISPLAY "Nota 1o Bimestre:"
DISPLAY "Nota 2o Bimestre:"
DISPLAY "Nota 3o Bimestre:"
DISPLAY "Nota 4o Bimestre:"
DISPLAY "Total das Notas.:"
DISPLAY "Media das Notas.:"
DISPLAY "Menssagem.......:"

DISPLAY "Informe o nome do aluno"

ACCEPT W-ALUNO

IF KEY-ESC >0
DISPLAY "Fim"
END-IF

IF W-ALUNO >= 0
DISPLAY "Campo obrigatorio"
END-IF

DISPLAY "Informe a primeira nota do aluno"
ACCEPT W-NOTA1
COMPUTE
W-NOTA1= W-NOTA1 + W-TOTAL-NOTAS
END-COMPUTE

DISPLAY "Informe a segunda nota do aluno"
ACCEPT W-NOTA2
COMPUTE
W-NOTA2 = W-NOTA2 + W-TOTAL-NOTAS
END-COMPUTE

DISPLAY "Informe a terceira nota do aluno"
ACCEPT W-NOTA3
COMPUTE
W-NOTA3 = W-NOTA3 + W-TOTAL-NOTAS
END-COMPUTE

DISPLAY "Informe a quarta nota do aluno"
ACCEPT W-NOTA3
COMPUTE
W-NOTA4= W-NOTA4 + W-TOTAL-NOTAS
END-COMPUTE

COMPUTE
W-MEDIA = W-TOTAL-NOTAS  / 4
END-COMPUTE
DISPLAY "WMEDIA"


IF W-MEDIA < 50
DISPLAY "REPROVADO!!!"
ELSE
IF W-MEDIA < 70
DISPLAY "RECUPERACAO!!!"
ELSE
DISPLAY "APROVADO!!!"
END-IF

STOP RUN.
END PROGRAM MEUPROGRAMA.
